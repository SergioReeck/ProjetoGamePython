# 🧮 Jogo de Matemática

Um jogo simples desenvolvido em Python para praticar operações matemáticas através do terminal. O jogador escolhe um nível de dificuldade, responde às operações geradas e acumula pontos a cada resposta correta.

## 🚀 Funcionalidades

* Escolha de níveis de dificuldade.
* Geração de operações matemáticas.
* Validação das respostas do usuário.
* Sistema de pontuação.
* Possibilidade de continuar jogando até desejar encerrar.

## 🛠️ Tecnologias Utilizadas

* Python 3

## 📋 Pré-requisitos

Antes de executar o projeto, certifique-se de ter instalado:

* Python 3.8 ou superior

## ▶️ Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/nome-do-repositorio.git
```

2. Acesse o diretório do projeto:

```bash
cd nome-do-repositorio
```

3. Execute a aplicação:

```bash
python main.py
```

## 🎮 Como Jogar

1. Escolha um nível de dificuldade entre 1 e 4.
2. Resolva a operação matemática exibida.
3. Dígite sua resposta e pressione Enter.
4. Receba sua pontuação atual.
5. Escolha continuar ou encerrar o jogo.

### Exemplo

```text
Informe o nível de dificuldade desejado [1, 2, 3, ou 4]: 2

Informe o resultado para a seguinte operação:
8 + 7

15

Você tem 1 ponto(s).

Deseja continuar no jogo? [1 - sim, 0 - não] 0

Você finalizou com 1 ponto(s).
```

## 📈 Sistema de Pontuação

* Cada resposta correta adiciona **1 ponto** à pontuação do jogador.
* A pontuação é mantida enquanto o jogador continuar na partida.
